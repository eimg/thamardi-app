• ၁၀ ဆံချည် = ၁ နှမ်း
  • ၆ နှမ်း = ၁ မုယော
  • ၄ မုယော = ၁ လက်သစ်
  • ၈ လက်သစ် = ၁ မိုက်
  • ၃ မိုက် = ၁ တောင်
  • ၄ တောင် = ၁ လံ
  • ၂ တောင် = ၁ ကိုက်
  • ၇ တောင် = ၁ တာ
  • ၂၀ တာ = ၁ ဥသဘ
  • ၂၀ ဥသဘ = ၁ ကောသ
  • ၄ ကောသ = ၁ ဂါဝုဒ်
  • ၄ ဂါဝုဒ် = ၁ ယူဇနာ
  • ၁၀ဝ၀ တာ = ၁ တိုင်
  • ၆ တိုင် နှင့် တာ ၄၀ဝ = ၁ ယူဇနာ
